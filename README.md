# Mina zkApp: Guess Race

Guess race zkapp registers in order first contest participant to find answer/secret for a game. 

## How to build

```sh
npm run build
```

## How to run tests

```sh
npm run test
npm run testw # watch mode
```

## How to run coverage

```sh
npm run coverage
```
